Instructions

Hello! This section will walk you through the main features of the app and help you get started quickly and efficiently.

Basic Functions

  1. Generating certificates.
  2. Calculating Cambridge Scores.
  3. Sending student scores to a Google Spreadsheet to build a database.

Basic Recommendations

  1. Double-check the spelling of your students' names, as they will appear on the final certificate. This is important because the certificate may be used at other educational institutions.
  2. Make sure you've entered the correct date.
  3. Ensure the exam level matches the score you're entering. Otherwise, you may get inaccurate results.
  4. Fill in all sections when entering test scores. If a student did not complete a section, enter 0 instead of leaving it blank.
  5. The teacher's feedback field has a 110-word limit, so keep comments concise.
  6. After clicking the Generate Certificate button, wait a few seconds for the PDF to generate, then review it carefully one more time.
  7. Check the Student DB to ensure the student's scores were uploaded successfully and there are no duplicates.
  8. Reload the page before starting a new certificate.

Hot Keys

Using hotkeys can speed up the process of filling in student information.

  • Arrow Down — moves the cursor to the Participant's Name field.
  • Digits 1–6 — select one of the session dates.
  • Tab — moves the cursor to the next input field.
  • S — Starters
  • M — Movers
  • F — Flyers
  • K — KET
  • P — PET
  • E — FCE
  • C — CAE
  • L — moves the cursor to the Listening score field.
  • Enter — generates a certificate.
