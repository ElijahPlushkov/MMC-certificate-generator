Instructions

Hello! This section will walk you through the main features of the app and help you get started quickly and efficiently.

Basic Functions

  1. Generating certificates.
  2. Calculating Cambridge Scores.
  3. Sending student scores to a Google Spreadsheet to build a database.

Basic Recommendations

  1. Double-check the spelling of your students' names, as they will appear on the final certificate. This is important, as the certificate may be used at other educational institutions.
  2. Make sure you've entered the correct date.
  3. Ensure the exam level matches the score you're entering. If not, you might get inaccurate results.
  4. When entering test scores, fill in all the sections. If a student didn't complete a particular part, enter a zero - don't leave the field blank.
  5. The teacher's feedback field has a 100-word limit, so keep your comments concise.
  6. After clicking the "Generate Certificate" button, wait a few seconds for the certificate to be generated. Then, review it one more time to catch any errors.

Hot Keys

Using hotkeys can speed up the process of filling in student info.


  • Arrow Down: moves the cursor to the Participant's name field.
  • Digits 1-6: select one of the session dates.
  • Tab: moves the cursor to the next input field.
  • S: Starters
  • M: Movers
  • F: Flyers
  • K: KET
  • P: PET
  • E: FCE
  • C: CAE
  • L: moves the cursor to the Listening score.
  • Enter: generates a certificate.
